    - (void) camel_case(str)

    Converts str to camelCase.

    Examples:

    _s.camelCase('foo-bar')      // => ('fooBar')
_s.camelCase('foo-bar-baz')  // => ('fooBarBaz')
_s.camelCase('foo:bar_baz')  // => ('fooBarBaz')
_s.camelCase('')             // => ('')
_s.camelCase('foo')          // => ('foo')
_s.camelCase('fooBar')       // => ('fooBar')

    - (void) swapcase(str)

    Returns a copy of str with uppercase alphabetic characters converted to lowercase and lowercase characters converted to uppercase. Note: case conversion is effective only in ASCII region.

    Examples:

    _s.swapcase("Hello")          // => "hELLO"
_s.swapcase("cYbEr_PuNk11")   // => "CyBeR_pUnK11"
