    - (void) wday(date)

    Returns an integer representing the day of the week, 0..6, with Sunday == 0.

    Examples:

      t = Time.now()   #=> 2007-11-20 02:35:35 -0600
  t.wday()         #=> 2
  t.sunday()      #=> false
  t.monday()      #=> false
  t.tuesday()     #=> true
  t.wednesday()   #=> false
  t.thursday()    #=> false
  t.friday()      #=> false
  t.saturday()    #=> false
