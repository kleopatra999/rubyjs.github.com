    - (Array<Number, Number>) gcdlcm(num, other)

    Returns an array; [int.gcd(int2), int.lcm(int2)].

    Examples:

    
  _n.gcdlcm(2,  2)                   #=> [2, 2]
  _n.gcdlcm(3, -7)                   #=> [1, 21]
  _n.gcdlcm((1<<31)-1, (1<<61)-1)    #=> [1, 4951760154835678088235319297]

    Returns:

    - (Number) lcm(num, other)

    Returns the least common multiple (always positive). 0.lcm(x) and x.lcm(0) return zero.

    Examples:

    
  _n.lcm(2,  2)                   #=> 2
  _n.lcm(3, -7)                   #=> 21
  _n.lcm((1<<31)-1, (1<<61)-1)    #=> 4951760154835678088235319297

    - (this) times(num, block)

    Iterates block int times, passing in values from zero to int - 1.

    If no block is given, an enumerator is returned instead.

    Examples:

    
  R(5).times(function(i) { R.puts(i) })
  # => 0 1 2 3 4
