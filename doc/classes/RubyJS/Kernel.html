    - (void) w(str)

    Equivalent to %w[] in Ruby

    Creates an R.Array of R.String for every word separated by space.

    Examples:

       R.w('foo bar   baz') # => ['foo', 'bar', 'baz']
   R.w('foo\nbar')      # => ['foo\nbar']
   R.w('')              # => ['']

    - (void) r(a, b, excluding)

    Shortcut for creating a R.Range.

    @alias #rng

    Examples:

       R.r(0,4)        # => (0..4)
   R.r(0,4, true)  # => (0...4)
